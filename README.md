# To-do Manager

개발 연습용 To-do Manager 프로그램

## 연습할것들

* TDD - 테스트 코드 작성 습관 들이기
* Git - 명령어 사용하는 연습
* Maven - 툴 사용법
* UML을 사용한 설계 - 개발 전 설계하는 연습
* Java의 함수형 문법
* 리팩토링
* 디자인패턴
* 가능하다면 프론트엔드 관련 기술 학습까지
